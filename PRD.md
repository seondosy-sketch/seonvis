## 1. 배경 및 문제 정의

미래사업팀은 매주 아래 내용을 포함한 주간/월간 보고서를 한글 문서로 작성한다.

- 수행 프로젝트 현황 (개찰 / 진행중 구분)
- 발주예상 프로젝트 목록
- 교육참가자 현황
- 프로젝트 일정 달력

기존 프로세스의 문제점:
- 매주 템플릿 HWPX 파일을 수동으로 열어 셀을 편집
- 데이터가 파일 단위로 분산되어 히스토리 추적 불가
- 담당자 부재 시 파일 위치·양식 파악에 시간 소요
- 발표일·개찰일 등 일정이 별도로 관리되어 누락 위험

## 4. 기능 요구사항

### 4-1. 수행 프로젝트 테이블 (필수)
- 개찰 / 진행중 그룹으로 분리 표시
- 인라인 편집: 용역명, 단장, 제출일, 발표/면접, 개찰일, 용역비(억원), 내용
- 행 추가 / 행 삭제
- 상태(개찰/진행중) 선택 가능

### 4-2. 발주예상 프로젝트 테이블 (필수)
- 인라인 편집: Project명, 발주청, 단장, 사업비, 발주월, 용역비, 내용
- 내용 필드 줄바꿈 지원

### 4-3. 교육참가자 입력 (필수)
- 책임 / 건축 / 토목 / 안전 / 기계 항목별 개별 입력

### 4-4. 데이터 저장 (필수)
- Supabase PostgreSQL에 현재 주차 기준으로 upsert
- 저장 버튼 클릭 시 전체 저장

### 4-5. HWPX 다운로드 (필수)
- **주간 HWPX**: 수행 프로젝트 + 발주예상 + 교육 내용 → weekly.hwpx 템플릿에 주입
- **월간 HWPX**: 수행 프로젝트 전체 → montly.hwpx 템플릿에 주입
- 원본 템플릿 서식 완전 보존 (XML 직접 조작 방식)
- 헤더 날짜 자동 업데이트

### 4-6. 일정 캘린더 (필수)
- 월간 그리드 캘린더
- 제출일(파랑), 발표/면접(주황), 개찰일(초록) 배지 표시
- 프로젝트명 함께 표시
- 월 이동 버튼

### 4-7. 사이드바 내비게이션 (필수)
- 3개 그룹: 사무 보조 / 업무 보조 / DB Bank
- 현재 활성 메뉴: 주간/월간보고
- 미완성 메뉴: 준비중 배지 표시
