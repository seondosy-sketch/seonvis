# 현장 현황 — 화면 설계

기존 Hub 디자인 시스템 그대로. **기술인 주소록(`/engineers`)과 같은 화면 문법**을 쓴다 —
요약 카드 + 검색/필터 툴바 + sticky 테이블 + 상세 모달 + 복사 토스트. 사용자가 두 화면을
오가며 같은 방식으로 쓸 수 있게 한다.

## 페이지 구조 (`/sites`)

```
┌ 헤더 바 ──────────────────────────────────────────────┐
│ 현장 현황                              [+ 현장 추가]     │
├ 요약 카드 ────────────────────────────────────────────┤
│ [전체 N] [진행 중 N] [준공 임박 N] [준공 완료 N]          │
├ 툴바 ─────────────────────────────────────────────────┤
│ [통합 검색_______] [법 구분▾][상태▾][지역▾][시공사▾]      │
│ [준공 시기▾] [정렬▾] [초기화]                            │
├ 목록 테이블 ───────────────────────────────────────────┤
│ 현장명│법구분│감리원│시공사│지역│현장연락처│착수일│준공예정│상태│ │
│ (현장명 좌측 고정 · 헤더 상단 고정 · 행 클릭 → 상세 모달)  │
└───────────────────────────────────────────────────────┘
```

## 목록 테이블

열: 현장명(좌측 sticky) · 법 구분(뱃지) · 감리원/담당자 · 시공사 · 지역 · 현장 연락처 ·
착수일 · 준공예정일 · 상태(뱃지) · 수정. 사무실주소·책임자HP·비고는 상세 모달에서만
(91건 × 열 과밀 방지). 즐겨찾기 ★ 첫 열.

- 현장명은 엑셀 원본의 줄바꿈을 공백으로 정규화해 표시, 말줄임 + title 툴팁.
- 감리원 셀은 여러 명 원본 텍스트 → 첫 이름 + "외 N명" 표시, title로 전체.
- 상태 뱃지 색: 진행 중(그린) / 준공 임박(앰버) / 준공 완료·중지(회색) / 착수 전(블루) /
  일정 미등록(연회색). 수동 지정된 상태는 뱃지에 점 표시(·)로 자동과 구분.
- 정렬: 현장명 가나다(기본) / 착수일순 / 준공예정일순 / 지역순 — 어떤 정렬이든
  진행 중(+임박) 현장 우선 규칙을 앞단에 유지.
- 비활성 현장은 "비활성 포함" 체크 시에만 표시, opacity 0.5.
- 91건 전체 렌더 (페이지네이션 불필요).

## 필터

- **법 구분**: 전체 / 건설기술진흥법 / 주택법 / 건축법 / 분리발주(전·통·소) —
  내부 값은 원본(건진법/주택법/건축법/전통소) 보존, 표시명만 매핑 (요구 6).
- 진행 상태(계산값 기준) / 지역(데이터에서 동적) / 시공사(동적) /
  준공 시기(전체·3개월 내·6개월 내·1년 내) / 초기화 버튼.

## 상세 모달 (행 클릭)

기술인 주소록 상세 모달과 동일 패턴(#111 헤더: 현장명 + 법구분·상태).
본문: 감리원(전체), 시공사, 착수일~준공예정일, 현장 연락처 [복사], 책임자 핸드폰 [복사],
현장주소 [복사], 사무실주소 [복사], 비고, 등록/수정일.
하단(쓰기 권한): [수정] [비활성화] [완전 삭제]. 복사 성공 시 토스트
("전화번호를 복사했습니다." / "현장주소를 복사했습니다.").

## 추가/수정 모달

필수: **현장명 · 법 구분**만 (요구 9 — 엑셀 데이터를 그대로 이관할 수 있어야 하므로
나머지는 전부 선택). 필드: 현장명 / 법 구분(4종 select) / 감리원(여러 줄 textarea) /
시공사 / 현장 연락처 / 책임자 핸드폰(자동 하이픈 — engineers의 formatPhone 재사용) /
현장주소(입력 시 지역 자동 추출 — extractRegion 재사용) / 지역 / 사무실주소 / 착수일 /
준공예정일 / 진행 상태(자동 + 수동 5종) / 비고 / ★ 즐겨찾기.
검증: 필수 누락 차단, 준공예정일 < 착수일 차단.

## 읽기 권한(menu_permissions: read)

+ 현장 추가, 수정/비활성/삭제, 즐겨찾기 토글 숨김. 검색·필터·복사·상세는 그대로.
